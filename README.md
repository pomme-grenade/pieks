# pieks

_A game about poking_

Play it at [https://pieks.rafa.ee](https://pieks.rafa.ee).

## Running in development mode

Requirements:

- npm
- python 3
- uvicorn
- fastapi

For the client:

```sh
cd frontend
npm install
npm run dev
```

For the server:

```sh
cd backend
uvicorn main:app --reload
```
